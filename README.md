# Student-Task-Manager
Students often struggle to keep track of their daily academic tasks, assignments, and deadlines. Existing solutions can be overly complex or distracting. This project aims to create a simple and efficient task management system for students.

# Student Task Manager

## Overview
This is a simple command-line based Student Task Manager developed in Java. It allows users to manage their daily tasks efficiently.
Help taken from GeeksforGeeks

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Persistent storage using file handling

## Technologies Used
- Java
- Object-Oriented Programming
- File Handling (Serialization)

## How to Run

1. Compile the program:
   javac TaskManager.java

2. Run the program:
   java TaskManager

## Usage
- Choose options from the menu
- Enter task details when prompted
- Tasks are automatically saved
